Features:
- Navbar with links to Home and Employee Form.
- Dashboard fetches data from https://jsonplaceholder.typicode.com/users.
- Displays ID, Name, and Email in a Bootstrap table.
- Employee form with fields: Name, Designation, Location, Salary (no POST required).
- Navigation handled via React Router.

To run:
1. Extract the zip.
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000 in your browser.
